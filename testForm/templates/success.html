NOTE : How to Fill Copy Paste Form ?

1. Just Copy Data and Paste it in the respective Data field.

2. If data is missing or not available in the paragraph write NO DATA (write in capital)in that respective Data field.

3. If there is more than one data for that respective field, write all data separated by comma. Do not give any space after or before the comma.
Example : Data1,Data2,Data3 so on...
